# Como subir no GitHub sem arquivos soltos

Este pacote foi organizado para subir apenas as pastas:

- backend/
- web/
- docs/

Não há arquivos soltos na raiz do pacote.

## No GitHub

1. Extraia este ZIP.
2. Entre no repositório Horizon-OEE-Site.
3. Clique em Add file > Upload files.
4. Arraste apenas as pastas backend, web e docs.
5. Confirme em Commit changes.

## Se já subiu arquivos soltos

Pode apagar da raiz do repositório os arquivos soltos de documentação/segurança que também existirem dentro de docs/ ou backend/.

Mantenha na raiz somente arquivos reais do site, como:

- index.html
- README.md
